## Overview

Data for an existing order can be fetched from the endpoint `/order/<orderId>/self`.
This endpoint will return a basic data set for placed orders that also includes links to the label endpoint and customer tracking page. The API for this endpoint can be considered stable, so it's preferable to use this endpoint instead of `/order/<orderId>` to retrieve order information.

```txt
GET https://api.porterbuddy-test.com/order/<orderId>/self
```

Example request
```bash
curl -X GET \
  https://api.porterbuddy-test.com/order/13218/self \
  -H 'x-api-key: <your_api_key>' \
  -H 'Accept: application/json'
```

## _Response fields_

| Parameter                 | Type                                                                    | Description                                                                                                                                  |
|---------------------------|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| id                        | String                                                                  | The Porterbuddy order ID                                                                                                                     |
| created                   | DateTime                                                                | The order creation date                                                                                                                      |
| deliverFrom               | DateTime                                                                | The start time of the delivery window                                                                                                        |
| deliverUntil              | DateTime                                                                | The end time of the delivery window                                                                                                          |
| pickupAddress             | [Resolved Address](https://developer.porterbuddy.com/#resolved-address) | The specified origin address for the order, usually the retailer's warehouse address                                                         |
| destinationAddress        | [Resolved Address](https://developer.porterbuddy.com/#resolved-address) | The order's recipient address (destination address)                                                                                          |
| pickupRegion              | String                                                                  | The region the order will be picked up (e.g. "OSLO", "BERGEN", ...). The list is non-exhaustive, so make sure to handle new values properly. |
| destinationRegion         | String                                                                  | The region the order will be delivered to                                                                                                    |
| recipientName             | String                                                                  | The name of the recipient                                                                                                                    |
| recipientPhoneNumber      | String                                                                  | The recipient's phone number                                                                                                                 |
| recipientPhoneCountryCode | String                                                                  | The recipient phone number's country code                                                                                                    |
| recipientEmail            | String                                                                  | The recipient's email address                                                                                                                |
| pickupName                | String                                                                  | The pickup name (the sender name specified in the order create request and printed on the label)                                             |
| pickupPhoneNumber         | String                                                                  | The sender's phone number                                                                                                                    |
| pickupPhoneCountryCode    | String                                                                  | The sender phone number's country code                                                                                                       |
| pickupEmail               | String                                                                  | The sender email as specified in the order create request - used for sender receipts if enabled                                              |
| pickupPin                 | String                                                                  | A 5-digit pin code from the parcel label                                                                                                     |
| deliveryType              | String                                                                  | The type of the delivery - currently always "SCHEDULED"                                                                                      |
| status                    | String                                                                  | The current order status, e.g. [received, delivered, cancelled]. The list is non-exhaustive, so make sure to handle new values properly.     |
| xxl                       | Boolean                                                                 | An indicator if the order is oversized/overweight                                                                                            |
| token                     | String                                                                  | The order token                                                                                                                              |
| parcels                   | List<[Parcel](https://developer.porterbuddy.com/#parcel)>               | The current parcel specification                                                                                                             |
| verifications             | [Verifications](https://developer.porterbuddy.com/#verifications)       | The verifications to perform on delivery                                                                                                     |
| orderReference            | String                                                                  | The external order reference as specified                                                                                                    |
| statusUpdatedAt           | DateTime                                                                | The timestamp of the last status update                                                                                                      |
| pickupDate                | Date                                                                    | The calculated pickup date, based on the specified delivery date                                                                             |
| pickedAt                  | DateTime                                                                | The timestamp the order was picked up by the courier                                                                                         |
| scannedAtPartner          | DateTime                                                                | The timestamp the order was scanned by the partner (retailer)                                                                                |
| scannedAtHub              | DateTime                                                                | The timestamp of the first time the order was scanned at a Porterbuddy hub                                                                   |
| deliveredAt               | DateTime                                                                | The timestamp the order was delivered at                                                                                                     |
| links                     | [OrderInfoLinks](https://developer.porterbuddy.com/#order-info-links)   | Links to the customer's order tracking page and label endpoint                                                                               |

## Order Info Links

| Parameter       | Type | Description                           |
|-----------------|------|---------------------------------------|
| labelInfo       | URL  | A link to the label info endpoint     |
| userInformation | URL  | A link to the recipient tracking page |
