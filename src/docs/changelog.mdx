## 5th January 2022
### Updates and improvements
- API - Added `pickupTime` to the `/order/<orderId>/self` endpoint which fetches the data for an order.

### Deprecation
- API - Deprecated `pickupDate` in the fetch order data endpoint `/order/<orderId>/self`.

## 10th December 2021
### Updates and improvements
- API - Added the `large` option in the `products` field when checking for Porterbuddy availability. This enables the provision of windows for oversized parcels. See [Porterbuddy large](/guides/porterbuddy-large).
- API - Deprecating `pickupDate` in the order creation response in favor of `pickupTime`.
