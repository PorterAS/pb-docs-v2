# Text replacement

Texts can also be replaced outside of the widget on elements with the data attribute `**porterbuddy-text**`**.**

## Example

> INSERT WIDGET

```html
<p>
  Custom text:
  <span data-porterbuddy-text="availabilityPostcodeSuccess">Placeholder</span>
</p>
<p>
  Custom text without pattern:
  <span
    data-porterbuddy-customtext="Custom text with postcode **{{postcode}}**"
  >
    Placeholder
  </span>
</p>
```
