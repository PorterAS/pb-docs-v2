Values set to `null` or `undefined` will be ignored and the default / previous value will be used for that particular text. If a text should be stopped from displaying, an emtpy string can be passed.
Variables can be used in text replacements to display changing values within the texts.

| **Key**                  | **Description**                                                                             |
| ------------------------ | ------------------------------------------------------------------------------------------- |
| remaining                | Countdown for last window of first day being available                                      |
| delivery                 | Delivery day for last window of first day being available                                   |
| postcode                 | Selected postcode                                                                           |
| location                 | Customer location                                                                           |
| price                    | Price of last delivery window of the first day                                              |
| countdown                | Countdown for last window of first day being available                                      |
| price_first              | Price of first delivery window                                                              |
| countdown_first          | Countdown for first window being available                                                  |
| delivery_first           | Delivery day for first delivery window                                                      |
| price_lowest_overall     | Price of cheapest delivery window over all windows                                          |
| countdown_lowest_overall | Countdown for cheapest delivery window over all windows being available                     |
| delivery_lowest_overall  | Delivery day for cheapest delivery window over all windows                                  |
| price_lowest_price       | Price of cheapest delivery window of first day                                              |
| countdown_lowest_price   | Countdown for cheapest delivery window of first day being available                         |
| delivery_lowest_price    | Delivery day for cheapest delivery window of first day                                      |
| selected_price           | Price of the currently selected window when available, otherwise the lowest available price |

## Example

> INSERT WIDGET

```html
<script>
  window.porterbuddy = {
    token: "y3wt37LqBsiLo62Jkx284XEdi4LzdX6pihZFwqYX",
    view: "availability",
    postalCode: "0278",
    language: "NO",
    text: {
      availabilitySuccessCountdown:
        "**{{remaining}}** remaining for this offer",
    },
  }
</script>
```
